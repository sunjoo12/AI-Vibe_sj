## 컬러 사용 가이드

### Primary — `#96B5B8` (뮤트 틸 세이지)
- **사용처**: 메인 배경, 히어로 섹션, 카드 배경, 네비게이션 배경
- **특징**: 채도를 낮춘 틸-그레이 계열로 부드럽고 몽환적인 감성 표현. 자연·힐링·감성 사진 계열 콘텐츠에 최적
- **주의**: 배경으로 사용 시 텍스트는 반드시 `--color-text-primary`(`#3D2820`) 이상의 명도 대비 확보

### Secondary — `#CC7058` (웜 코랄)
- **사용처**: CTA 버튼, 헤드라인 포인트, 강조 배지, 호버 언더라인
- **특징**: 차가운 틸 배경과 따뜻한 코랄의 보색 대비로 즉각적 시선 집중. 꽃의 생명력과 에너지를 상징
- **주의**: 전체 UI의 15~20% 이내로 제한. 과용 시 차분한 감성 톤이 무너짐

### Accent — `#D4A060` (웜 앰버 골드)
- **사용처**: 아이콘 포인트, 별점, 뱃지, 호버 글로우 효과
- **특징**: 꽃 수술의 황금빛을 연상시켜 프리미엄·따뜻함·자연 소재 이미지 강화
- **주의**: 소량 포인트로만 사용 (5% 이내). 코랄과 동시 사용 시 상호 충돌 주의

### Sage Green — `#7A9E88` (세이지 그린)
- **사용처**: 태그, 자연·에코 관련 배지, 식물 카테고리 아이콘
- **특징**: 꽃줄기와 잎사귀의 색으로 자연스러운 생명감 부여. 틸 계열 배경과 유사 색조로 조화
- **주의**: 코랄·앰버 포인트와 함께 쓸 때 자연스러운 트리아드 조화 형성

### Blush Pink — `#E0A8A4` (블러시 핑크)
- **사용처**: 호버 오버레이, 소프트 배경 포인트, 이미지 필터 효과
- **특징**: 꽃잎의 연분홍 블러 효과를 재현. 감성적이고 로맨틱한 인상 추가
- **주의**: 배경 전용 또는 오버레이 전용으로 사용. 텍스트 컬러로 단독 사용 금지 (대비율 부족)

### Background Lavender — `#C0B2BC` (라벤더 그레이)
- **사용처**: 섹션 구분 배경, 카드 보조 배경, 모달 오버레이
- **특징**: 보랏빛이 살짝 감도는 그레이로 틸 배경과 부드러운 대비. 몽환적 분위기 연출

## 반응형 고려사항

### 다크모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경 — 딥 올리브 틸로 전환, 몽환 감성 유지 */
    --color-bg-primary: #1A2628;
    --color-bg-secondary: #243035;
    --color-bg-lavender: #2A2430;

    /* 텍스트 — 크림 화이트 계열로 반전 */
    --color-text-primary: #F4EDE8;
    --color-text-secondary: #D4C4BC;
    --color-text-muted: #A08A84;

    /* 코랄은 밝기 유지, 다크 배경에서도 대비 확보 */
    --color-secondary: #E08068;
    --color-button-primary: #E08068;
    --color-button-hover: #CC7058;

    /* 테두리·하이라이트 어둡게 */
    --color-border: #3A2C28;
    --color-highlight: #242020;

    /* 링크 밝게 */
    --color-link: #96B5B8;
    --color-link-hover: #B8D0D4;
  }
}
```

### 그라디언트 배경 (핵심 활용법)
```css
/* 히어로 섹션 — 라벤더에서 틸로 흐르는 꽃잎 블러 그라디언트 */
.hero-section {
  background: linear-gradient(
    135deg,
    #C0B2BC 0%,
    #F4EDE8 40%,
    #96B5B8 100%
  );
}

/* 오버레이 글로우 효과 — 꽃잎 블러 연출 */
.hero-overlay {
  background: radial-gradient(
    ellipse at 30% 50%,
    rgba(204, 112, 88, 0.15) 0%,
    rgba(244, 237, 232, 0.05) 60%,
    transparent 100%
  );
}

/* 다크모드 그라디언트 */
@media (prefers-color-scheme: dark) {
  .hero-section {
    background: linear-gradient(
      135deg,
      #2A2430 0%,
      #1A2628 50%,
      #243035 100%
    );
  }
}
```

### 접근성 (WCAG 2.1 AA 기준)
| 조합 | 대비율 | 판정 |
|------|--------|------|
| `#3D2820` on `#F4EDE8` | **9.8:1** | ✅ AAA 통과 |
| `#3D2820` on `#96B5B8` | **5.2:1** | ✅ AA 통과 |
| `#FFFFFF` on `#CC7058` | **3.4:1** | ✅ AA 통과 (굵은 텍스트·대형 아이콘) |
| `#FFFFFF` on `#96B5B8` | **2.1:1** | ⚠️ 본문 텍스트 단독 사용 금지 |
| `#3D2820` on `#C0B2BC` | **6.1:1** | ✅ AA 통과 |

> **주의**: 틸(`#96B5B8`) 및 라벤더(`#C0B2BC`) 배경 위 흰색 텍스트는 대비율 부족으로 **사용 금지**.
> 밝은 계열 배경 전체에 반드시 딥 웜 브라운(`#3D2820`) 텍스트 사용.
