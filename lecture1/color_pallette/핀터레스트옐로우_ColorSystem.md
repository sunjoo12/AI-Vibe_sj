## 컬러 사용 가이드

### Primary — `#D4A017` (골든 머스타드 옐로우)
- **사용처**: 메인 배경, 히어로 섹션, 카드 배경, 네비게이션 배경
- **특징**: 채도 높은 머스타드 톤으로 따뜻하고 생동감 있는 감성 표현. 자연·플로럴·무드보드 계열 콘텐츠에 최적
- **주의**: 배경으로 사용 시 텍스트는 반드시 `--color-text-primary`(`#4A3418`) 이상의 명도 대비 확보

### Secondary — `#E8963C` (웜 허니 앰버)
- **사용처**: CTA 버튼, 헤드라인 포인트, 강조 배지, 호버 언더라인
- **특징**: 머스타드 배경 위에서 한 톤 더 짙은 앰버로 자연스러운 시선 집중. 노을·꿀빛의 온기를 상징
- **주의**: 전체 UI의 15~20% 이내로 제한. 과용 시 단조로운 단색 느낌으로 무너짐

### Accent — `#F5E27A` (소프트 레몬 옐로우)
- **사용처**: 아이콘 포인트, 하이라이트 배지, 호버 글로우 효과
- **특징**: 레몬·백합 꽃잎의 밝은 빛을 연상시켜 상큼하고 청량한 포인트 강화
- **주의**: 소량 포인트로만 사용 (5~8% 이내). 화이트 배경 위 단독 사용 시 대비 부족 주의

### Leaf Green — `#8FA55C` (리프 그린)
- **사용처**: 태그, 자연·플로럴 관련 배지, 잎사귀 카테고리 아이콘
- **특징**: 백합·마리골드의 줄기와 잎사귀 색으로 옐로우 배경과 보색 대비를 이루며 생명감 부여
- **주의**: 머스타드·앰버 포인트와 함께 쓸 때 자연스러운 유사색 조화 형성

### Daisy White — `#FFFDF7` (데이지 화이트)
- **사용처**: 카드 배경, 오버레이, 텍스트 강조 박스
- **특징**: 데이지 꽃잎·저널 종이의 화이트로 옐로우 톤의 채도를 낮춰 시각적 휴식 공간 제공
- **주의**: 배경 전용으로 사용. 짙은 옐로우 배경 위에 얹을 때만 대비 효과 발생

## 반응형 고려사항

### 다크모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경 — 딥 올리브 브라운으로 전환, 따뜻한 감성 유지 */
    --color-bg-primary: #2A2210;
    --color-bg-secondary: #3A2F14;
    --color-bg-pale: #332A18;

    /* 텍스트 — 크림 아이보리 계열로 반전 */
    --color-text-primary: #FBF3DE;
    --color-text-secondary: #E0D4B0;
    --color-text-muted: #A89468;

    /* 앰버는 밝기 유지, 다크 배경에서도 대비 확보 */
    --color-secondary: #F0A550;
    --color-button-primary: #F0A550;
    --color-button-hover: #E8963C;

    /* 테두리·하이라이트 어둡게 */
    --color-border: #4A3C1E;
    --color-highlight: #2A2410;

    /* 링크 밝게 */
    --color-link: #D4A017;
    --color-link-hover: #F0DFA0;
  }
}
```

### 그라디언트 배경 (핵심 활용법)
```css
/* 히어로 섹션 — 크림에서 머스타드로 흐르는 플로럴 그라디언트 */
.hero-section {
  background: linear-gradient(
    135deg,
    #FBF3DE 0%,
    #F0DFA0 40%,
    #D4A017 100%
  );
}

/* 오버레이 글로우 효과 — 노을빛 연출 */
.hero-overlay {
  background: radial-gradient(
    ellipse at 30% 50%,
    rgba(232, 150, 60, 0.15) 0%,
    rgba(251, 243, 222, 0.05) 60%,
    transparent 100%
  );
}

/* 다크모드 그라디언트 */
@media (prefers-color-scheme: dark) {
  .hero-section {
    background: linear-gradient(
      135deg,
      #332A18 0%,
      #2A2210 50%,
      #3A2F14 100%
    );
  }
}
```

### 접근성 (WCAG 2.1 AA 기준)
| 조합 | 대비율 | 판정 |
|------|--------|------|
| `#4A3418` on `#FBF3DE` | **10.1:1** | ✅ AAA 통과 |
| `#4A3418` on `#D4A017` | **5.6:1** | ✅ AA 통과 |
| `#FFFFFF` on `#E8963C` | **2.6:1** | ⚠️ 본문 텍스트 단독 사용 금지 (굵은 대형 텍스트만 허용) |
| `#4A3418` on `#F0DFA0` | **7.4:1** | ✅ AAA 통과 |
| `#FFFFFF` on `#D4A017` | **2.2:1** | ⚠️ 본문 텍스트 단독 사용 금지 |

> **주의**: 머스타드(`#D4A017`) 및 허니 앰버(`#E8963C`) 배경 위 흰색 텍스트는 대비율 부족으로 **본문 텍스트 사용 금지** (대형 굵은 텍스트·아이콘만 허용).
> 밝은 계열 배경 전체에 반드시 딥 엄버 브라운(`#4A3418`) 텍스트 사용.
